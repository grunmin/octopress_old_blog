unit 4

gpg的使用
+ 流程：
由user1产生密钥对，公钥导出给需要向user1传送文件的主机user2。
gpg —gen-key
gpg -K 查看私钥
gpg -k 查看公钥
gpg —export —armor key-id -o > file.key
user2导入公钥加密文件，传送给user1。
gpg —import file.key
gpg -e -armor -r key-id file
user1运用私钥解密文件，用私钥前需输入密码。
gpg -d file > file_without_encrypt
